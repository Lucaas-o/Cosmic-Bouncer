Use these shortcuts to control the simulation:

  • P: Toggle Pause/Resume
  • S: Cycle Speed (0.5x, 1x, 2x)
  • A: Toggle Attraction
  • G: Toggle Gravity
  • B: Cycle Ball Count (10, 30, 50, 100)
  • C: Go to this Info / Contributing page

üòÉ How to Contribute

  1. Fork the repository on GitHub.
  2. Clone your fork locally: git clone https://github.com/alicelee2735/Cosmic-Bouncer.git
  3. Create a feature branch: git checkout -b feature/your-feature
  4. Make your changes.
  5. Test in multiple browsers (Chrome, Firefox, etc.).
  6. Commit with descriptive messages: git commit -m "Add your feature"
  7. Push to your fork: git push origin feature/your-feature
  8. Create a Pull Request back to this repo.
