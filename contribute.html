Contribute to Cosmic Bouncer

Welcome to the Cosmic Bouncer project! We're excited to have you help make this simulation even better.

üòÉ How to Contribute

  1. Fork the repository on GitHub.
  2. Clone your fork locally: git clone https://github.com/Lucaas-o/Cosmic-Bouncer.git
  3. Create a feature branch: git checkout -b feature/your-feature
  4. Make your changes.
  5. Test in multiple browsers (Chrome, Firefox, etc.).
  6. Commit with descriptive messages: git commit -m "Add your feature"
  7. Push to your fork: git push origin feature/your-feature
  8. Create a Pull Request back to this repo.
